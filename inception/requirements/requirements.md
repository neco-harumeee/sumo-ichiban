# 要件定義書 — TRAINING LOCK

## インテント分析サマリー

| 項目 | 内容 |
|------|------|
| **ユーザーリクエスト** | AWS ハッカソン向け「人をダメにする」サービス。トレーニング愛好家の「鍛える以外の意思決定」を AI が肩代わりし、集中時間を守る |
| **リクエストタイプ** | 新規プロジェクト（Greenfield） |

## サービス概要

**TRAINING LOCK** は、トレーニング愛好家が「3時間消える」と宣言すると、Amazon Bedrock を中心とした AI エージェントが以下を自動実行するサービス。

1. **社会的シールド** — 仕事メールや家族への連絡を AI が代理送信し、トレーニング中の不在を自然に隠蔽
2. **モチベ反論機能** — サボりそうになったユーザーに Bedrock が圧強めの反論を生成し、Amazon Polly が音声で送りつける
3. **Amazon 物流連携** — トレーニングギアを自動発注し「ぽちっとしておきました」と事後報告

## 機能要件

### FR-01: トレーニング宣言 UI
- ユーザーが Web アプリ上で「トレーニング開始」「終了予定時刻」を入力できる
- シンプルなハッカソンデモ用画面（認証なし）
- 入力後、AI エージェントが自動起動する

### FR-02: 社会的シールド（メール自動返信）
- トレーニング中に受信したメールに対し、Bedrock が文脈に応じた自然な返信文を生成
- 「現在外出対応中です」等、状況に合わせた文面を自動送信
- **実装方針**: コードはモック（実際には送信しない）。デモ動画では実際に Gmail 送信を行う
- GitHub 公開コードには実際の送信処理を含めない

### FR-03: 社会的シールド（家族への連絡）
- 家族に対して「買い物して帰ります」等の連絡を適切なタイミングで自動送信
- Bedrock が送信タイミングと文面を判断
- **実装方針**: FR-02 と同様にモック実装

### FR-04: モチベ反論機能
- ユーザーがサボりそうなタイミング（宣言時刻を過ぎた場合等）を検知
- Bedrock が「雨の日は空気抵抗が減ります」等の圧強めの反論テキストを生成
- Amazon Polly で音声ファイル（MP3）を生成
- 音声ファイルをメール / LINE に添付して送信

### FR-05: Amazon 物流連携
- トレーニングギア（シューズ等）を Amazon PA-API 経由で検索・発注
- 発注後「シューズ、ぽちっとしておきました。」とユーザーに通知
- **実装方針**: コードはモック（PA-API 呼び出しはスタブ）。デモでは実際に PA-API を使用

### FR-06: AI エージェント制御
- Amazon Bedrock（Claude 3 系または Titan、動けば OK）を使用
- 各機能（返信文生成・反論生成・発注判断）を Bedrock が統合的に処理
- AWS Lambda でサーバーレス実行

## 非機能要件

### NFR-01: AWS サービス構成
- **Amazon Bedrock** — テキスト生成（返信文・反論文）
- **Amazon Polly** — 反論テキストの音声合成（MP3 出力）
- **AWS Lambda** — サーバーレス処理基盤
- **Amazon API Gateway** — Web UI からのリクエスト受付
- **Amazon S3** — 生成した音声ファイルの一時保存
- **Amazon DynamoDB** — トレーニングセッション状態管理（任意）

### NFR-02: 認証・セキュリティ
- 認証なし（ハッカソンデモ用 PoC）
- セキュリティ拡張ルール: **スキップ**（PoC のため）
- GitHub 公開コードには実際のメール送信・PA-API 呼び出しを含めない

### NFR-03: テスト方針
- プロパティベーステスト: **スキップ**（PoC のため）
- 基本的な動作確認テストのみ実施

### NFR-04: デモ・提供形態
- Web アプリ（ブラウザ動作、PC・スマホ対応）
- デモ形式: スライド + 動画デモ（録画）
- ライブデモは不要

### NFR-05: 開発スコープ
- ハッカソン期間内に完結する実装量
- 全部盛り（FR-01〜FR-05 すべて）だが、モック実装で工数を抑える
