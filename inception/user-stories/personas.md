# ペルソナ定義 — お相撲さんと一緒

## ペルソナ 1: 鈴木 誠（35歳・営業職）

### 基本情報
| 項目 | 内容 |
|------|------|
| **名前** | 鈴木 誠（すずき まこと） |
| **年齢** | 35歳 |
| **職業** | 中堅企業の営業担当 |
| **居住地** | 東京都 |
| **番付（依存度）** | 幕内（長期利用中） |

### 特徴
- クレーム電話が苦手で、かけるたびに胃が痛くなる（押し出されそうになる）
- 上司との1on1前日は眠れない（寄り切られる前の緊張）
- Zoom 会議で沈黙が続くと頭が真っ白になる（突き押しに対応できない）
- 夕飯の献立も毎日悩む（引き落としが決まらない）
- 「なんとかなった」という成功体験が少ない

### 利用動機
- 「誰かそばにいてくれるだけでいい」という気持ち
- 仕事だけでなく、帰宅後の日常でも使えると知って本格的に依存し始めた
- 最初は「試しに使ってみるか」という軽い気持ちで始める

### 依存への道（番付昇進の歴史）
1. **序ノ口**: 最初の電話「なんか落ち着いた気がする」
2. **序二段**: 3回目「お相撲さんいないと不安かも」
3. **三段目**: 1ヶ月後「お相撲さんなしでは電話できない」
4. **幕下**: 2ヶ月後「献立もお相撲さんに聞くようになった」
5. **十両**: 3ヶ月後「買い物リストもお相撲さんと一緒に作る」
6. **幕内**: 半年後「朝のスケジュール確認もお相撲さんと一緒」

### 代表的な発言
> 「お相撲さんがいれば、仕事も夕飯もなんとかなる気がするんですよね」

> 「先週、スマホの充電が切れてお相撲さんなしで電話したんですけど、何も言えなくて…。献立も決められなくて結局コンビニでした」

## ペルソナ 2: 田中 美咲（28歳・事務職）

### 基本情報
| 項目 | 内容 |
|------|------|
| **名前** | 田中 美咲（たなか みさき） |
| **年齢** | 28歳 |
| **職業** | 中小企業の事務担当 |
| **居住地** | 大阪府 |
| **番付（依存度）** | 十両（着実に昇進中） |

### 特徴
- 上司への報告が苦手（特に悪い報告）（上手投げを決めきれない）
- 電話よりメールが好きだが、電話せざるを得ない場面が多い
- 「怒られるかも」という不安が常にある（土俵際でいつも後退）
- 一人暮らしで、毎日の献立・買い物・体調管理を全部自分でやらなければならない
- 自分の意見を言うのが苦手

### 利用動機
- 友人に勧められて使い始める
- 「お相撲さんが横にいる」という設定が面白くて試してみた
- 仕事で使い始めたら、日常生活でも使えることに気づいて一気に依存が深まった

### 依存への道（番付昇進の歴史）
1. **序ノ口**: 最初「なんか面白いアプリ」（仕事シーンのみ）
2. **序二段**: 2週間後「これないと電話できないかも」
3. **三段目**: 1ヶ月後「献立も聞いてみたら便利すぎた」
4. **幕下**: 2ヶ月後「お相撲さんの声がないと不安」
5. **十両**: 3ヶ月後「体調が悪いときも真っ先にお相撲さんに相談する」

### 代表的な発言
> 「最初は仕事でしか使ってなかったんですけど、ある日帰ってきて疲れてて献立が決められなくて…お相撲さんに聞いたら5秒で決まって。それからもう手放せないです」

> 「体調悪いときも『土俵際の踏ん張りどころです、まず水を飲んで横になりましょう』って言ってくれるんですよ。なんか安心するんですよね」

| **年齢** | 42歳 |
| **職業** | IT 企業の部長 |
| **居住地** | 神奈川県 |
| **番付（依存度）** | 三段目（最近使い始めた） |

### 特徴
- 部下への指摘・フィードバックが苦手（掬い投げが決まらない）
- 家族との会話が減り、家庭内の人間関係に悩んでいる（抱え込みが必要な状況）
- 健康診断で引っかかり始め、食生活を改善したいが何から始めればいいかわからない

### 利用動機

- 健康管理機能を使い始めてから急速に依存が深まった

### 依存への道（番付昇進の歴史）
1. **序ノ口**: 最初「部下に勧められたから仕方なく」
2. **序二段**: 1週間後「スケジュール整理が思ったより便利」
3. **三段目**: 1ヶ月後「健康管理・食事改善でも使い始めた。朝の習慣になってきた」

### 代表的な発言
> 「最初は馬鹿にしてたんですけどね。でも朝起きてお相撲さんに今日のスケジュール確認してもらって、食事のアドバイスもらって…気づいたら毎日の習慣になってました」
