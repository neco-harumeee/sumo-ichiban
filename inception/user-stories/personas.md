# ペルソナ定義 — お相撲さんと一緒

## ペルソナ 1: 鈴木 誠（35歳・営業職）

### 基本情報
| 項目 | 内容 |
|------|------|
| **名前** | 鈴木 誠（すずき まこと） |
| **年齢** | 35歳 |
| **職業** | 中堅企業の営業担当 |
| **居住地** | 東京都 |

### 特徴
- クレーム電話が苦手で、かけるたびに胃が痛くなる
- 上司との面談前日は眠れない
- Zoom 会議で沈黙が続くと頭が真っ白になる
- 「なんとかなった」という成功体験が少ない

### 利用動機
- 「誰かそばにいてくれるだけでいい」という気持ち
- 問題を解決してほしいわけではなく、ただ安心したい
- 最初は「試しに使ってみるか」という軽い気持ちで始める

### 依存への道
1. 最初の電話：「なんか落ち着いた気がする」
2. 3回目：「お相撲さんいないと不安かも」
3. 1ヶ月後：「お相撲さんなしでは電話できない」
4. 3ヶ月後：「どすこいが無いと会議に出られない」

### 代表的な発言
> 「お相撲さんがいれば、なんとかなる気がするんですよね」

> 「先週、スマホの充電が切れてお相撲さんなしで電話したんですけど、何も言えなくて…」

## ペルソナ 2: 田中 美咲（28歳・事務職）

### 基本情報
| 項目 | 内容 |
|------|------|
| **名前** | 田中 美咲（たなか みさき） |
| **年齢** | 28歳 |
| **職業** | 中小企業の事務担当 |
| **居住地** | 大阪府 |

### 特徴
- 上司への報告が苦手（特に悪い報告）
- 電話よりメールが好きだが、電話せざるを得ない場面が多い
- 「怒られるかも」という不安が常にある
- 自分の意見を言うのが苦手

### 利用動機
- 友人に勧められて使い始める
- 「お相撲さんが横にいる」という設定が面白くて試してみた
- 使ってみたら本当に落ち着いた

### 依存への道
1. 最初：「なんか面白いアプリ」
2. 2週間後：「これないと電話できないかも」
3. 2ヶ月後：「お相撲さんの声がないと不安」
4. 半年後：「お相撲さんに相談してから決める」という習慣が定着
