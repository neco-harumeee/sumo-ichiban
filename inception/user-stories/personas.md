# ペルソナ定義 — TRAINING LOCK

## ペルソナ 1: 田中 剛（たなか つよし）

### 基本プロフィール
| 項目 | 内容 |
|------|------|
| **年齢** | 34歳 |
| **職業** | IT 系会社員（フルリモート勤務） |
| **居住** | 東京都内、一人暮らし |
| **トレーニング歴** | 3年（ランニング + 筋トレ） |

### 特徴・背景
- 週4〜5回のトレーニングを「人生の軸」と位置づけている
- リモートワークで仕事とプライベートの境界が曖昧になりがち
- 「今日は雨だから…」「メール返してから…」と言い訳を作りがち
- 家族（実家の母）や職場からの連絡が集中する時間帯にトレーニングしたい
- Amazon でトレーニングギアを衝動買いする習慣がある

### 目標
- 週のトレーニング計画を100%達成したい
- 「鍛える時間」を誰にも邪魔されたくない
- 言い訳を自分の外側に追い出したい

### フラストレーション
- 「ちょっといい？」という連絡でトレーニングが中断される
- 罪悪感から集中できない（「メール返してないな…」）
- モチベーションが下がったとき、自分を奮い立たせる言葉が見つからない

### TRAINING LOCK との関係
- **主要ユーザー**: サービスを直接操作し、トレーニング宣言を行う
- **受益者**: 社会的シールド・モチベ反論・物流連携すべての恩恵を受ける
- **デモシナリオの主役**: ハッカソン審査員に見せるすべての操作を行う人物
