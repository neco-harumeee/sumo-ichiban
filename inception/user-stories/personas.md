# ペルソナ定義 — お相撲さんと一緒

# ペルソナ 1: 鈴木 誠（35歳・営業職）

## 基本情報

| 項目 | 内容 |
|------|------|
| **名前** | 鈴木 誠（すずき まこと） |
| **年齢** | 35歳 |
| **職業** | 中堅企業の営業担当 |
| **居住地** | 東京都 |
| **番付（依存度）** | 幕内（長期利用中） |

---

## 特徴

- クレーム電話や謝罪連絡が極端に苦手
- 上司との 1on1 や重要な会議の前は強い緊張を感じる
- 会話の沈黙や「どう返せばいいかわからない瞬間」に弱い
- 日常の小さな判断でも不安になりやすい
- 「一人でちゃんと乗り切れた」という感覚を持てた経験が少ない

---

## 利用動機

- 「会話をうまく進めたい」よりも、「一人じゃない感覚」が欲しかった
- 最初は仕事の電話サポート用途として試しに使い始めた
- 徐々に、私生活の不安や苦手な会話でも使うようになった
- “うまく話せるようになる”というより、“安心して話し始められる”感覚に依存していった

---

## 依存への道（番付昇進の歴史）

| 番付 | 状態 |
|------|------|
| **序ノ口** | 最初の電話「なんか落ち着いた気がする」 |
| **序二段** | 3回目「お相撲さんいないと不安かも」 |
| **三段目** | 1ヶ月後「お相撲さんなしでは電話できない」 |
| **幕下** | 2ヶ月後「苦手な連絡をするとき、隣にいてもらわないと無理になった」 |
| **十両** | 3ヶ月後「役所や病院に電話するとき、必ず一緒にいてもらうようになった」 |
| **幕内** | 半年後「重要な話や謝罪を、一人では切り出せなくなった」 |

---

## 代表的な発言

> 「お相撲さんが隣にいるだけで、ちゃんと話せる気がするんですよね」

> 「先週、スマホの充電が切れてお相撲さんなしで電話したんですけど、何も言えなくて…。結局、用件を伝えられずに切っちゃいました」

# ペルソナ 2: 佐藤 健太（42歳・管理職）

## 基本情報

| 項目 | 内容 |
|------|------|
| **名前** | 佐藤 健太（さとう けんた） |
| **年齢** | 42歳 |
| **職業** | IT企業の部長 |
| **居住地** | 神奈川県 |
| **番付（依存度）** | 三段目（最近使い始めた） |

---

## 特徴

- 部下への指摘・フィードバックが苦手（掬い投げが決まらない）
- 家族との会話が減り、家庭内の人間関係に悩んでいる（抱え込みが必要な状況）
- 忙しすぎて重要な対話を後回しにしがち（寄り切りが必要）

---

## 利用動機

- 部下に「使ってみてください」と勧められて渋々始めた

---

## 依存への道（番付昇進の歴史）

| 番付 | 状態 |
|------|------|
| **序ノ口** | 最初「部下に勧められたから仕方なく」 |
| **序二段** | 1週間後「部下へのフィードバック前にいてもらうと気が楽になった」 |
| **三段目** | 1ヶ月後「苦手な会話の前には必ず呼ぶようになった」 |

---

## 代表的な発言

> 「最初は馬鹿にしてたんですけどね。でも部下との面談とか、妻とちゃんと話す前とか、お相撲さんが横にいるだけで全然違うんですよ。気づいたら毎日の習慣になってました」
