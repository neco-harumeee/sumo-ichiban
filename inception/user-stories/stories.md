# ユーザーストーリー — 相撲一番（SUMO ICHIBAN）

## エピック一覧

| シーン | お相撲さんの技 | 相撲技 | 説明 |
|----|---------|--------|------|
| E-01 | 場面選択・起動 | 仕切り | お相撲さんを呼び出す |
| E-02 | 仕事サポート | 押し出し・寄り切り・突き押し・掬い投げ・上手投げ | 仕事場面の精神的サポート |

## ストーリー一覧

### E-01: 場面選択・起動（仕切り）

#### US-01: 場面を選んでお相撲さんを呼ぶ
**As a** 困った場面に直面しているユーザーとして  
**I want to** 今から始まる場面を選択したい  
**So that** お相撲さんに四つに組んでもらえる

**受け入れ基準**:
- [ ] Web アプリで仕事シーン（クレーム電話・面談・会議・相談・報告）と日常シーン（献立・買い物・体調・スケジュール・人間関係）から選択できる
- [ ] 選択後、お相撲さんのキャラクターが画面に表示される
- [ ] 「どすこい、四つに組みました」という起動メッセージが表示される
- [ ] 選択した場面に対応する相撲技名が表示される（例：クレーム電話→「押し出し」）

#### US-02: お相撲さんの同席を終了する
**As a** ユーザーとして  
**I want to** 場面が終わったらお相撲さんを退場させたい  
**So that** セッションが記録される

**受け入れ基準**:
- [ ] 「終了」ボタンでセッションを終了できる
- [ ] 終了時に「よく耐えました。また土俵に呼んでください」等のメッセージが表示される
- [ ] セッション時間と依存度スコア（番付）が更新される

### E-02: 仕事サポート（押し出し・寄り切り・突き押し）

#### US-03: 電話・面談中に助言メッセージを受け取る
**As a** クレーム電話・上司面談中のユーザーとして  
**I want to** 緊張を検知したときに助言を受け取りたい  
**So that** 場面を最後まで乗り切れる

**受け入れ基準**:
- [ ] マイク入力から音声を取得し Amazon Transcribe でリアルタイム解析する
- [ ] 緊張ワード（「申し訳」「すみません」「困ります」等）を検知したら助言を表示する
- [ ] 助言メッセージは相撲技・相撲用語を含む（「押し出しで前に進みましょう」「まわしは取られておりません」等）
- [ ] Bedrock が状況に応じた実用的な助言テキストを生成する（謝罪の言い回し、交渉の切り返し等）

#### US-04: お相撲さんの声で助言される
**As a** 電話・面談中のユーザーとして  
**I want to** お相撲さんの音声で助言を聞きたい  
**So that** より安心感を得られる

**受け入れ基準**:
- [ ] Amazon Polly でお相撲さんらしい音声を生成する
- [ ] 生成した音声（MP3）を S3 に保存する
- [ ] ブラウザで音声を再生する（イヤホン推奨）

#### US-05: 沈黙が続いたときに助け舟を出してもらう
**As a** 電話・面談中に言葉が出なくなったユーザーとして  
**I want to** 沈黙を検知して助言を受け取りたい  
**So that** 会話を再開するきっかけをもらえる

**受け入れ基準**:
- [ ] 3秒以上の沈黙を検知したら助言を表示する
- [ ] 「間がありますな。土俵際の踏ん張りどころです」等のメッセージを表示する
- [ ] 具体的な切り返しフレーズを提案する（「『少々お時間をいただけますか』と言いましょう」等）

#### US-06: 会議の隅にお相撲さんを常駐させる
**As a** Zoom 会議中のユーザーとして  
**I want to** 画面の隅にお相撲さんを表示したい  
**So that** 会議中も安心感を得られる

**受け入れ基準**:
- [ ] ブラウザの隅（右下）にお相撲さんのウィジェットが常駐する
- [ ] 最小化・最大化できる
- [ ] 他のウィンドウの上に重なって表示される（常に最前面）
- [ ] 発言タイミングの助言を表示する（「突き押しで今が発言のタイミングです」等）

### E-03: 日常生活サポート（引き落とし・叩き込み・割り出し）

#### US-07: 夕飯の献立を提案してもらう
**As a** 帰宅後に疲れて献立が決められないユーザーとして  
**I want to** 冷蔵庫の食材から献立を提案してもらいたい  
**So that** 迷わず夕飯の準備を始められる

**受け入れ基準**:
- [ ] テキストまたは音声で「冷蔵庫にある食材」を入力できる
- [ ] Bedrock が食材から実用的な献立を提案する（引き落とし技で迷いを断ち切る）
- [ ] 調理時間・難易度・栄養バランスを考慮した提案をする
- [ ] 「引き落としで献立を決めました」等の相撲用語を含むメッセージを表示する
- [ ] 提案に対して「他の案は？」と聞き直せる

#### US-08: 買い物リストを管理してもらう
**As a** 買い忘れが多いユーザーとして  
**I want to** 買い物リストを音声・テキストで管理したい  
**So that** 買い忘れなく効率よく買い物できる

**受け入れ基準**:
- [ ] 音声またはテキストでリストに追加・削除できる
- [ ] 「叩き込みで必要なものを押さえました」等のメッセージを表示する
- [ ] カテゴリ別（野菜・肉・調味料等）に整理して表示する
- [ ] 前回の買い物リストから「よく買うもの」を提案する

#### US-09: スケジュールを整理してもらう
**As a** 予定が込み入って混乱しているユーザーとして  
**I want to** 今日・今週のスケジュールを整理してもらいたい  
**So that** 優先順位をつけて行動できる

**受け入れ基準**:
- [ ] テキストまたは音声で予定を入力できる
- [ ] Bedrock が優先順位・時間配分を提案する（割り出し技で整理する）
- [ ] 「割り出しでスケジュールを整理しました」等のメッセージを表示する
- [ ] 過密スケジュールを検知したら「土俵際です。何かを削りましょう」と警告する

#### US-10: 人間関係の悩みを相談する
**As a** 人間関係で悩んでいるユーザーとして  
**I want to** 悩みを聞いてもらいアドバイスをもらいたい  
**So that** 次の行動を決められる

**受け入れ基準**:
- [ ] テキストまたは音声で悩みを入力できる
- [ ] Bedrock が共感しつつ実用的なアドバイスを返す（抱え込み技で受け止める）
- [ ] 「抱え込みで受け止めました。一緒に考えましょう」等のメッセージを表示する
- [ ] センシティブな内容（深刻なメンタルヘルス）を検知した場合は専門機関への相談を促す

### E-04: 健康・体調サポート（土俵際の踏ん張り）

#### US-11: 体調不良時にアドバイスをもらう
**As a** 体調が優れないユーザーとして  
**I want to** 症状を伝えて対処法を教えてもらいたい  
**So that** 適切な対処ができる

**受け入れ基準**:
- [ ] テキストまたは音声で症状を入力できる
- [ ] Bedrock が一般的な対処法を提案する（土俵際の踏ん張りで体調を立て直す）
- [ ] 「土俵際の踏ん張りどころです。まず水を飲んで横になりましょう」等のメッセージを表示する
- [ ] 重篤な症状を示すキーワードを検知した場合は必ず「医療機関への受診を推奨」を表示する
- [ ] 医療診断は行わない旨を明示する

#### US-12: 食事・健康習慣のアドバイスをもらう
**As a** 健康改善を目指しているユーザーとして  
**I want to** 食事・運動・睡眠のアドバイスをもらいたい  
**So that** 無理なく健康習慣を続けられる

**受け入れ基準**:
- [ ] 現在の生活習慣をテキストで入力できる
- [ ] Bedrock が実践的な改善提案をする
- [ ] 「四股を踏むように、毎日少しずつ体を鍛えましょう」等の相撲用語を含む提案をする
- [ ] 無理な目標設定を検知したら「一気に横綱を目指さず、序ノ口から始めましょう」と助言する

### E-05: 依存度の可視化（番付昇進）

#### US-13: 依存度（番付）を確認する
**As a** 継続利用しているユーザーとして  
**I want to** 自分の依存度（番付）を確認したい  
**So that** どれだけ依存しているかを把握できる（そして気にしなくなる）

**受け入れ基準**:
- [ ] ダッシュボードに「お相撲さん依存度」を番付形式で表示する
- [ ] 利用回数・利用時間・サポート要求回数からスコアを算出する
- [ ] スコアに応じてお相撲さんのランク（序ノ口〜横綱）が上がる
- [ ] 仕事シーン・日常シーン別の利用内訳を表示する

#### US-14: 依存度に応じたメッセージを受け取る
**As a** 依存度が高くなったユーザーとして  
**I want to** 番付に応じたメッセージを受け取りたい  
**So that** 依存していることを自覚しつつも離れられない

**受け入れ基準**:
- [ ] 番付が上がるほど「もう離れられませんな」等のメッセージを表示する
- [ ] 「お相撲さん無しで決断した回数: 0回」等の統計を表示する
- [ ] 横綱到達時に「土俵は私のものです」という特別メッセージを表示する

#### US-15: 利用履歴を確認する
**As a** ユーザーとして  
**I want to** 過去の利用履歴を確認したい  
**So that** どれだけ頼ってきたかを振り返れる

**受け入れ基準**:
- [ ] 日付・場面（仕事/日常）・セッション時間・サポート回数の履歴を表示する
- [ ] 「初めてお相撲さんに四つに組んでもらった日」を表示する
- [ ] 仕事シーンと日常シーンの利用比率をグラフで表示する
