# ユーザーストーリー — 相撲一番（SUMO ICHIBAN）

## コンセプト

「一人では切り出せない会話」に、お相撲さんが四つに組んでくれる。

相撲一番（SUMO ICHIBAN）は、  
仕事・日常・人間関係の “気まずさ” や “不安” を、  
お相撲さんが精神的に支えてくれる伴走型サポートアプリ。

ユーザーは助言そのものよりも、  
「隣に誰かがいる安心感」に徐々に依存していく。

# E-01: 場面選択・起動（仕切り）

## US-01: 場面を選んでお相撲さんを呼ぶ

### User Story

**As a** 困った場面に直面しているユーザー  
**I want** 今から始まる場面を選択したい  
**So that** お相撲さんに四つに組んでもらえる

### 受け入れ基準

- 仕事シーン（クレーム電話・面談・会議・報告）を選択できる
- 日常シーン（家族・LINE返信・病院予約・人間関係）を選択できる
- 選択後、お相撲さんが画面に現れる
- 「どすこい、四つに組みました」と表示される
- 場面に応じた技名が表示される
  - クレーム電話 → 「押し出し」
  - 会議 → 「突き押し」
  - 家族との会話 → 「土俵際」

---

## US-02: お相撲さんの同席を終了する

### User Story

**As a** ユーザー  
**I want** お相撲さんとのセッションを終了したい  
**So that** 利用履歴と依存度が記録される

### 受け入れ基準

- 「終了」ボタンで退出できる
- 終了時に労いメッセージを表示する
  - 「よく耐えました」
  - 「また土俵に呼んでください」
- セッション時間が保存される
- 番付スコアが更新される

# E-02: 仕事サポート（押し出し・寄り切り・突き押し）

## US-03: 緊張時に助言を受け取る

### User Story

**As a** クレーム電話や面談中のユーザー  
**I want** 緊張した時に助言を受け取りたい  
**So that** 最後まで会話を続けられる

### 受け入れ基準

- マイク入力をリアルタイム解析する
- 緊張ワードを検知する
  - 「申し訳ありません」
  - 「困ります」
  - 「すみません」
- 状況に応じた助言を表示する
  - 「押し出しで前へ出ましょう」
  - 「まわしは取られておりません」
- 実用的な切り返し例を提案する

---

## US-04: お相撲さんの声で励ましてもらう

### User Story

**As a** 面談・電話中のユーザー  
**I want** 音声で励ましてほしい  
**So that** 安心感を得られる

### 受け入れ基準

- お相撲さん風の音声を再生できる
- ブラウザから再生できる
- イヤホン利用を想定する
- 「どすこい、大丈夫です」などの音声が流れる

---

## US-05: 沈黙時に助け舟を出してもらう

### User Story

**As a** 会話が止まってしまったユーザー  
**I want** 沈黙時に助言してほしい  
**So that** 会話を再開できる

### 受け入れ基準

- 3秒以上の沈黙を検知する
- 助言メッセージを表示する
  - 「土俵際ですな」
  - 「ここが踏ん張りどころです」
- 会話再開フレーズを提案する
  - 「少々お時間をいただけますか」
