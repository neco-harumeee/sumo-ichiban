## ストーリー一覧

### E-01: 場面選択・起動

#### US-01: 場面を選んでお相撲さんを呼ぶ
**As a** 緊張しやすいユーザーとして  
**I want to** 今から始まる嫌な場面を選択したい  
**So that** お相撲さんに同席してもらえる

**受け入れ基準**:
- [ ] Web アプリで「クレーム電話」「上司との面談」「Zoom会議」「苦手な相談」「気まずい報告」から選択できる
- [ ] 選択後、お相撲さんのキャラクターが画面に表示される
- [ ] 「どすこい、参りました」という起動メッセージが表示される

#### US-02: お相撲さんの同席を終了する
**As a** ユーザーとして  
**I want to** 場面が終わったらお相撲さんを退場させたい  
**So that** セッションが記録される

**受け入れ基準**:
- [ ] 「終了」ボタンでセッションを終了できる
- [ ] 終了時に「よく耐えました」等のメッセージが表示される
- [ ] セッション時間と依存度スコアが更新される

### E-02: クレーム電話同行

#### US-03: 電話中に励ましメッセージを受け取る
**As a** クレーム電話中のユーザーとして  
**I want to** 緊張を検知したときに励ましを受け取りたい  
**So that** 電話を最後まで乗り切れる

**受け入れ基準**:
- [ ] マイク入力から音声を取得し Amazon Transcribe でリアルタイム解析する
- [ ] 緊張ワード（「申し訳」「すみません」「困ります」等）を検知したら励ましを表示する
- [ ] 励ましメッセージは相撲用語を含む（「落ち着いております」「まわしは取られておりません」等）
- [ ] Bedrock が状況に応じた励ましテキストを生成する

#### US-04: お相撲さんの声で励まされる
**As a** 電話中のユーザーとして  
**I want to** お相撲さんの音声で励ましを聞きたい  
**So that** より安心感を得られる

**受け入れ基準**:
- [ ] Amazon Polly でお相撲さんらしい音声を生成する
- [ ] 生成した音声（MP3）を S3 に保存する
- [ ] ブラウザで音声を再生する（イヤホン推奨）

#### US-05: 沈黙が続いたときに助け舟を出してもらう
**As a** 電話中に言葉が出なくなったユーザーとして  
**I want to** 沈黙を検知して励ましを受け取りたい  
**So that** 会話を再開するきっかけをもらえる

**受け入れ基準**:
- [ ] 3秒以上の沈黙を検知したら励ましを表示する
- [ ] 「間がありますな」「ここは沈黙の横綱です」等のメッセージを表示する

### E-03: 会議同席

#### US-06: Zoom 会議の隅にお相撲さんを常駐させる
**As a** Zoom 会議中のユーザーとして  
**I want to** 画面の隅にお相撲さんを表示したい  
**So that** 会議中も安心感を得られる

**受け入れ基準**:
- [ ] ブラウザの隅（右下）にお相撲さんのウィジェットが常駐する
- [ ] 最小化・最大化できる
- [ ] 他のウィンドウの上に重なって表示される（常に最前面）

#### US-07: 長時間ミュートを検知して励ましを受け取る
**As a** 会議でミュートが続いているユーザーとして  
**I want to** 長時間ミュートを検知して励ましを受け取りたい  
**So that** 会議への参加意欲を保てる

**受け入れ基準**:
- [ ] 5分以上のミュート継続を検知したら励ましを表示する
- [ ] 「長丁場ですな」「まだ土俵にいます」等のメッセージを表示する

#### US-08: 早口・詰まりを検知して励ましを受け取る
**As a** 会議中に緊張して早口になったり言葉に詰まったりするユーザーとして  
**I want to** 発話パターンの変化を検知して励ましを受け取りたい  
**So that** 気づかないうちにサポートされる

**受け入れ基準**:
- [ ] Transcribe の文字起こし結果から発話速度（単語/秒）を計算する
- [ ] 通常より30%以上速い発話（早口）を検知したら励ましを表示する
- [ ] 「えーと」「あの」「その」等のフィラーワードが3回以上連続したら励ましを表示する
- [ ] 「焦っておりますな」「まわしを締め直しましょう」等のメッセージを表示する

### E-04: 依存度の可視化

#### US-09: 依存度スコアを確認する
**As a** 継続利用しているユーザーとして  
**I want to** 自分の依存度スコアを確認したい  
**So that** どれだけ依存しているかを把握できる（そして気にしなくなる）

**受け入れ基準**:
- [ ] ダッシュボードに「お相撲さん依存度」スコアを表示する
- [ ] 利用回数・利用時間・緊張検知回数からスコアを算出する
- [ ] スコアに応じてお相撲さんのランク（序ノ口〜横綱）が上がる

#### US-10: 依存度に応じたメッセージを受け取る
**As a** 依存度が高くなったユーザーとして  
**I want to** 依存度に応じたメッセージを受け取りたい  
**So that** 依存していることを自覚しつつも離れられない

**受け入れ基準**:
- [ ] 依存度が上がるほど「もう離れられませんな」等のメッセージを表示する
- [ ] 「お相撲さん無しで電話した回数: 0回」等の統計を表示する

#### US-11: 利用履歴を確認する
**As a** ユーザーとして  
**I want to** 過去の利用履歴を確認したい  
**So that** どれだけ頼ってきたかを振り返れる

**受け入れ基準**:
- [ ] 日付・場面・セッション時間・励まし回数の履歴を表示する
- [ ] 「初めてお相撲さんに頼った日」を表示する
